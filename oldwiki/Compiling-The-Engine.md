# Compiling the Engine

## Requirements
1. Install the latest stable version of [Haxe](https://haxe.org/download/)  
2. Install [Git](https://git-scm.com/downloads) and leave all options at default  
3. Run the [install-haxelibs](setup/install-haxelibs.bat) script to install required libraries  
4. Install [Visual Studio 2022](https://visualstudio.microsoft.com/es/downloads/)

## Building

To compile the engine, run:

```

lime test [platform]

```
