Installation

Supported platforms

  • **Windows**: Nightly builds ship with dependencies bundled.
  • **Linux**: Use the AppImage or ensure FFMPEG libraries are available.
  • **Android**: Sideload the APK and grant storage permissions on first launch.

Checklist

  • [ ] Downloaded the correct archive.
  • [ ] Extracted to a writable directory.
  • [ ] Tested the console with `debugMode = true`.

Need help? Add a comment below and include your OS plus the build date you downloaded.
